# Changelog

## [1.0.0] - 2026-03-27

### Added

- Initial release of the Home Services Logbook add-on.
- Bundles [Home Services Logbook](https://github.com/orionshock/HomeServicesLogbook) as a Home Assistant add-on.
- Home Assistant Ingress support for seamless UI access.
- Persistent data storage via the `/data` add-on volume (SQLite database + file uploads).
- Configurable options: `trust_upstream_auth` and `allow_actor_override`.
- Multi-architecture support: `amd64`, `aarch64`, `armv7`, `armhf`, `i386`.
